問題6

希望可以手把手一步步說明該如何利用圖形介面 menu 建立這個場景, 過程中有那些要點必須特別注意?

1. 重要的 Webots 介面功能

  1. 場景樹（Scene Tree）

    • 使用 右鍵 > Add New 來添加新物件。
    • 可隨時刪除 (Delete) 或重新調整物件屬性。

  2. 3D 視圖（3D View）

    • 右鍵拖動可旋轉視角，滾輪可放大/縮小。

  3. 屬性面板（Property Editor）

    • 調整 size、translation 和 physics 參數。

  4. 運行與測試（Simulation Controls）

    • 播放  啟動遊戲。
    • 暫停  可隨時調整場景。
    • 停止  重設到初始狀態。

2. 注意事項

  1. 確保籃球有 physics 設置，否則它無法彈跳或移動。
  2. 籃框與桌面必須為靜態 Solid，避免被球撞倒。
  3. 發射裝置的 LinearMotor 需要足夠的 maxForce 才能推動球。
  4. 避免物件重疊，否則會導致模擬錯誤（例如球穿透桌面）。
  5. 適時存檔 File > Save World As，避免進度遺失。
